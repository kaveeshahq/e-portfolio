## 📸 How to Add Photos

### Step 1: Upload Photos to Folders

1. Navigate to the corresponding folder in `public/assets/photos/`
2. Upload your image files (JPG, PNG, WebP recommended)
3. Name files descriptively

**Example naming:**

```
activities/01-community-assessment-setup.jpg
activities/02-ipaq-administration.jpg
activities/03-socio-demographic-questionnaire.jpg
activities/04-height-weight-measurement.jpg
activities/05-waist-hip-circumference.jpg
activities/06-handgrip-strength-test.jpg
activities/07-key-informant-interview.jpg
```

## 💡 Photo Tips

✅ **Do:**

- Use high-quality images (JPG/PNG)
- Ensure good lighting and visibility
- Include people when possible (with consent)
- Capture activities from multiple angles
- Use consistent naming convention

❌ **Don't:**

- Use blurry or low-contrast images
- Upload huge files (resize if > 5MB)
- Use overly compressed images
- Mix naming conventions

## 📊 Typical File Sizes

- Single photo: 2-4 MB (after optimization)
- Video clip: 50-200 MB (depending on duration)
- **Total ideal size:** < 500 MB for all assets
